# 🌍 GotW — Guardian of the Worlds

**Персональный архив и конструктор миров для художников**

GotW — это веб-приложение, созданное специально для художников-иллюстраторов, концепт-дизайнеров и worldbuilding-энтузиастов. Оно решает проблему фрагментированного хранения творческих идей, объединяя визуальные референсы, текстовые описания и связи между элементами мира в единой системе.

---

## 🚀 Демо

**Публичная версия:** [https://gotw.onrender.com](https://gotw.onrender.com)

> *Примечание:* Сервер работает на бесплатном тарифе Render.com. При отсутствии активности он может «засыпать». Первый запрос после пробуждения может занимать до 30-40 секунд.

---

## ✨ Основные возможности

| Функция | Описание |
|:---|:---|
| **Управление мирами** | Создание, редактирование и удаление миров (вселенных) с обложками |
| **Гибкие элементы** | Создание персонажей, локаций, предметов с произвольными JSON-полями (`custom_fields`) |
| **Система типов** | Пользовательские категории элементов (персонаж, локация, предмет, заклинание и т.д.) |
| **Связи между элементами** | Создание отношений (друг, враг, семья, учитель и др.) с указанием силы связи |
| **Галереи изображений** | Загрузка и привязка изображений к любым сущностям через универсальную галерею |
| **Аутентификация** | Регистрация, вход, восстановление пароля, личный профиль |
| **Экспорт в Google Sheets** | Выгрузка данных о мирах и элементах в Google Sheets одной командой |
| **Админ-панель** | Управление всеми данными через встроенную админку Django |
